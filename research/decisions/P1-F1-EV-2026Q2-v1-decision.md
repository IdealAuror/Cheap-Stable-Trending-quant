# 决策 — P1-F1-EV-2026Q2-v1

## 决策类型
放弃方向（因子重设计）

## 决策
F1-EV<0 因子（v1 口径：二值 `factor_ev_negative` + 连续 `-ev`）在 AllA 样本"形式上"通过 Gate 1 七条中的六条，但判定为**伪通过**：不进回测，直接进 Phase 1.1 重设计。CS300/CSI500 两样本作废（V2 因 `ev_ext=ev` bug 与 V1 同值，待新脚本补跑）。

### 2. 三个结构性硬伤

**硬伤 A：二值因子在所有样本结构性失效**
- CSI300 EV<0 占比 12%（17/143），CSI500 **0%**（0/242，因子无分布），AllA 仅 1.5%（18/1196）。
- 二值 IC t 三样本均 < 2（1.63 / 0.25 / 0.53）。
- 根因：A 股长期高估值，"现金 > 市值+有息负债"的 Graham net-net 极端口径几乎没有股票满足。非参数问题，放弃。

**硬伤 B：连续 `-ev` 本质是小市值代理**
- `ev = market_cap*1e8 + 有息负债 - cash`，数值上 `market_cap` 项绝对压倒后两项，`signal=-ev` 排序 ≈ `signal=-market_cap`。
- 证据链：
  - 市值中性化后 AllA IC t 从 2.78 腰斩至 1.32（失效）——超一半信号是市值暴露。
  - 分市值档：小盘 t=4.94（强）、中盘 t=3.04、大盘 t=0.55（无效）——因子只在小盘有效，典型"市值效应伪装成基本面因子"。
  - 三样本敏感性强反转：CSI300/CSI500（大盘为主）均失效，仅 AllA（小盘占比最高）显著。
- 结论：AllA 的 Gate 1 通过，本质是 A 股小盘 alpha 的功劳，非"净现金"本身。进回测只会复现小盘股 alpha，违背项目 value-investing 定位。

**硬伤 C：金融股污染（数据卫生缺陷）**
- CSI300 与 AllA 的 `ev` 最小值完全相同（-2.32e12），指向同一只大盘金融股。
- 金融股"cash_equivalents"含经营性头寸（客户存款相关），非 Graham 意义净现金，会让 ev 产生荒谬离群值。
- 当前 `_get_stock_pool` 与过滤条件**无任何行业剔除**。金融股系统性扎堆 Q1（最低 ev），污染多头组收益。
- AllA 的 Q1 年化 93.9% 中金融股贡献占比无法分辨，因子"有效性"可能部分来自金融板块暴露。

### 3. V2 bug 记录
- `P1-F1-EV-2026Q2-v1-ic_analysis-standalone.py` 原 265-266 行 `df['ev_ext'] = df['ev']`，导致 V2≡V1，所有 V2 结果作废。
- 已修复：`ev_ext` 改用 `monetary_funds + financial_assets_held_for_trading` 扩展现金口径；V1/V2 通过 `use_ext` 参数切换。修复后未重跑全量。

## 后果（对后续 Phase 的影响）

### 立即行动（Phase 1.1，非可选）
1. **金融股剔除**：`_get_stock_pool` 或过滤层剔除银行/保险/证券（申万一级行业），重跑 AllA-V1 取干净基线。若 -2.32e12 离群值消失且 IC 仍显著，因子才有真腿。
2. **因子重设计**：
   - 废弃 `factor_ev_negative`（二值，结构性样本不足）
   - 废弃连续 `-ev`（绝对值，被市值主导）
   - 新增 `net_cash_yield = (cash_equivalents - interest_bearing_debt) / market_cap`（比值，天然去市值共线，横截面可比）
3. **决定性验证门槛**：新因子必须满足"市值中性化后 IC NW-t ≥ 2"才算真因子，否则仍是 size proxy。这是 v1 与 v2 因子版本的分水岭。

### 暂缓
- V2（壳价值剔除变体）不单独跑。等 `net_cash_yield` 落地后，壳价值剔除作为样本稳健性检验附带跑一次。
- CS300/CSI500 的 V2 用修复后脚本补跑，仅作记录，不影响 Phase 1.1 方向。

### 为什么继续迭代仍合理（放弃方向重启条件，第六机制）
F1 的经济逻辑（净现金公司抗风险、有回购/分红能力）本身没错，错在"绝对值 + 二值"的工程实现不适配 A 股估值结构。标准化为比值口径后，经济逻辑可被重新检验——若 `net_cash_yield` 在市值中性化后仍显著，说明"净现金"确有跨市值增量信息，值得进回测；若仍失效，则证明 A 股的"净现金"信号本质就是 size，届时彻底放弃 F1 方向。

### 致命缺陷：Q1-Q5 无区分度（AllA）

AllA 五分组年化收益几乎一条平线：Q1=50.7%、Q2=49.6%、Q3=48.4%、Q4=52.4%、Q5=52.1%。做多 Q5 与做空 Q1 间几乎无价差。IC 的统计显著性（t=2.29）来自少数极端值的 rank 相关，因子在分组层面无经济区分力——典型"假显著"。

### 其他发现

1. **-ev 在 CSI300 反转**：剔除金融股后 -ev 的 t 从 v1 的 1.39 变成 -0.14，证明 v1 里 CSI300 的信号基本是金融股撑的。AllA 的 -ev 仍 t=2.73，小盘效应在剔除金融股后依然主导。
2. **注册制断点全面失效**：三样本均前段强后段弱（CSI300 前 t=3.68→后 0.68；AllA 前 t=2.73→后 0.97）。net_cash_yield 是注册制前（壳价值时代）有效，注册制后失效——非稳定因子。

### 最终判定：F1 方向终止

net_cash_yield 的标准化方向正确（CSI300 IC t 从 -ev 的 -0.14 提到 2.44），但未通过决定性门槛，且暴露更深层问题：
- 市值中性化后失效 → 净现金信号本质仍与 size 绑定
- Q 分组无区分度 → 经济意义薄弱，IC 是极端值驱动的假显著
- 注册制后失效 → 非稳定因子，特定历史阶段产物

**F1-EV<0 / net_cash 方向彻底放弃。** 第六机制重启条件不再满足：标准化为比值后，市值中性化门槛仍未通过，证明 A 股的"净现金"信号本质上就是 size 效应的代理，不存在跨市值的增量信息。后续因子研发转向与市值正交的其他价值维度（如盈利质量、分红行为）。

### 突破与缺陷并存

**突破**：AllA 市值中性化 t=2.52，是三轮验证（F1-v1/v2/F2-A）中**第一个过门槛**的因子。证明 A 股存在与 size 正交的价值因子信息，F1 的失败是因子定义问题而非方向问题。数学框架预测的"稳态化降低 size 共线"被验证。

**缺陷 1：Q 分组方向倒挂**。三个样本均 Q1(最贵)收益最高、Q5(最便宜)最低。IC 正但 Q 反向，说明因子中段有效、两端反向：极便宜=价值陷阱，极贵=成长溢价。线性 IC/Q 框架不适合捕获此非线性。

**缺陷 2：全样本 t 未到 2.0**（AllA 仅 1.40）。显著性来自 AllA 横截面宽度，单样本不够稳。

**缺陷 3：注册制后全面失效**。三样本均前段强后段弱（AllA 前 t=1.50→后 0.48），与 F1-v2 同模式。

### 核心结论：当期 EP 全面碾压稳态 EP

数学预测（稳态化降 size 共线 → 中性化 IC 更高）被数据推翻。原因：
1. 市场对当期盈利的定价偏差本身就是信号，稳态化把它平滑掉了
2. 3年窗口引入 survivorship bias + 数据缺失 + 信息滞后，副作用 > 收益
3. 市值中性化已直接剥离 size，不需要靠稳态化降 β——中性化是更干净的手段

**方法论教训**：数学分析用于排除明显错误（F1 的 -ev 分子主导 P，判断正确），但不能用于预测哪个因子更优。正确做法是先跑所有候选的无过滤基线，让数据说话，再用数学解释数据。

### 决策：进回测

**ep_spot（当期 EP）通过 Gate 1，进回测。** 理由：
1. 决定性门槛大幅通过（t=3.41/2.94，非临界），是四轮验证中最强的信号
2. Q 倒挂和全样本 t 未到 2.0 是 A 股成长溢价的系统性特征，不否定因子存在；回测用 IC 加权全截面选股可规避极端值
3. 注册制后失效是所有 A 股价值因子共性，回测区间覆盖前后两段可观察完整表现
4. IC 方法论已兑现价值（拦住 F1 伪因子，确认 F2 真因子），继续 IC 迭代边际收益低，回测能提供换手/回撤/容量等增量信息

回测口径：AllA 全 A 样本，季度调仓，**ep_spot = net_profit / (market_cap*1e8)**，IC 加权选股（非 Q5 简单多头），金融股剔除保留。

### 结论

**F2-EP 纯因子版本为当前最优基线**：年化超额 6.5%，正 Alpha，因子选股能力验证通过。

**PE 锚定择时失败**，三个原因：
1. 沪深300 PE 和全A选股组合不对齐（持仓偏小盘价值，PE锚的是大盘估值）
2. 牛市减仓代价 > 熊市减仓收益（A 股涨幅集中，PE 13→15 就开始减仓，错过主升浪）
3. 日频调仓摩擦成本高

PE 择时对 ETF 轮动有效（quant-insights/04 验证），但对个股选股策略不对症。**择时和选股是两个维度，强行叠加适得其反。**

### 回撤问题的正确解法

单因子策略高回撤（51.6%）是固有特征，不是择时能解决的。正解是**多因子组合降波**：
- F2-EP 是价值因子，成长股牛市时回撤大
- 需找低相关因子对冲：F3 股息率（现金流稳定性）、F4 质量 ROE（成长股牛市表现好）、低波动因子

### 回测结果（三策略对比）

| 策略 | 总收益 | Alpha | Sharpe | 最大回撤 |
|------|--------|-------|--------|----------|
| **F2-EP 单因子** | **366%** | **0.07** | **0.34** | **51.6%** |

### 最终判定：F4-ROE 方向终止

ROE 在 A 股作为独立排序因子是**负 alpha**（-0.01，收益 86% 跑输基准 121%）。加 ROE 到 EP 上全方位劣化（收益 366%→187%，回撤 51.6%→57.1%）。

**"ROE 限制太宽"不是问题本质。** 根因：
1. 高 ROE = 高估值 = 贵，A 股惩罚贵股（与 EP 信号冲突）
2. ROE 单独无选股能力（IC 未过门槛）
3. 加严 ROE 门槛只会更差（F2-EP 质量过滤 ROE≥3% 已验证砍 81% 样本信号崩溃）

ROE 的唯一可能用法是过滤条件而非排序因子，但已验证是负增量。**F4-ROE 方向终止。** F4 IC 脚本保留作记录，回测脚本已删除。

### 降回撤方向复盘

F2-EP 回撤 51.6% 的降回撤尝试全部失败：
- PE 锚定择时：失败（牛市减仓代价 > 熊市减仓收益）
- 叠加 ROE 质量因子：失败（负 alpha，全方位劣化）

单因子高回撤是固有特征。正解是找与 EP **低相关**的下行保护因子做多因子组合，不是择时，不是叠加质量因子。

## Phase 1.4 启动（F5-LowVol 低波动率）

### 方向选择理由

低波动率因子（LowVol）是降回撤的理想候选：
1. **与 EP 低相关**：EP 选便宜股（价值），LowVol 选稳健股（防御），选股池不同
2. **下行保护**：低波动股熊市跌幅小，正好补 EP 成长股牛市回撤大的短板
3. **A 股低波动溢价**：A 股散户占比高，过度追捧高波动股，低波动溢价比美股更显著
4. **数据现成**：历史价格直接算波动率，不踩财务字段名坑

### 因子定义

- 主信号：`vol_60d = -std(过去60交易日日收益率)`，取负使低波动=高信号=IC>0有效
- 对照信号：`vol_120d = -std(过去120交易日日收益率)`，验证窗口敏感性
- 有效观测下限：`lookback * 0.5`（避免新股/长期停牌样本不足）

### Size 共线风险（决定性测试）

低波动率因子天然与市值共线：小盘股波动大，LowVol 信号与 ln(market_cap) 强相关。市值中性化后 IC NW-t≥2 是决定性门槛——若未过则 LowVol 也是 size proxy（与 F1-EV 同病）。

银行股天然低波动，保留金融股剔除（sw_l1 严格相等匹配）避免信号被银行主导，同时与 F2-EP 口径一致便于后续组合。

### Q 分组倒 U 型特征

F5 的 Q 分组不是单调递增，而是**倒 U 型**：绝对收益最高在 Q2/Q3（中段波动），Q5（最低波动）绝对收益平庸但风险调整后收益最优。

- Q5 vs Q1：月均 0.0282 vs 0.0270，Gate 1 第2条形式通过
- Q5 的价值在**波动降 30%（28.5% vs 40.5%）、夏普提升 50%（1.39 vs 0.93）**，不在绝对收益
- 这是"低波动率之谜"在 A 股的体现：散户过度追捧高波动股，低波动溢价存在但非单调

### 多空夏普 FAIL 的解释

多空夏普 0.05 衡量 Q5-Q1 对冲组合。Q5 约等于 Q1（0.0282 vs 0.0270），对冲零收益导致夏普接近 0，这是数学必然。与 F2-EP 的 Q 倒挂 FAIL 同理：多空夏普衡量的是对冲收益，而我们要的是**多头组合**（IC 加权全截面选股），不是做空一端。低波动的价值是**风险管理属性**（降波动/提夏普），不是 alpha 属性（高绝对收益）。

3. 断点后增强——注册制后仍有效，结构性溢价

2. F2+F5 双因子组合：等权 z-score 合成，验证是否降回撤（F5 的终极使命）

## Phase 2 回测结果（F5-LowVol，2026-06-27）

### F5 单因子回测：失败

| 指标 | F5-LowVol 单因子 | F2-EP 单因子 | 基准 |
|------|-----------------|-------------|------|
| 总收益 | **2.81%** | 366% | 121% |
| Alpha | **-0.04** | 0.07 | — |
| Sharpe | **-0.34** | 0.34 | — |
| 最大回撤 | 35.44% | 51.6% | — |

F5 作为独立策略彻底失败：12 年只赚 2.81%，负 alpha 负夏普。

**失败原因：z-score 加权与倒 U 型因子不适配。**
1. IC 验证 Q5（800 只等权）月均 0.0282 → 年化 39.7%，但回测取最低波动的前 50 只（Q5 中最极端 6%），落在倒 U 型的极端端——高速公路/港口/公用事业等大盘滞涨股
2. z-score 加权进一步把权重堆在波动率最低的几只上，等于在"收益平庸区"极端下注
3. F2-EP 用 z-score 有效是因为 EP 是单调信号；F5 的 Q 分组倒 U 型，z-score 选最低波动恰好选到收益最差段

### 关键教训：日波动率 ≠ 最大回撤

之前判断"F5 提供下行保护（Q5 波动降 30%）"是**错误的**，混淆了两个概念：
- IC 验证的"波动率"= 日收益率标准差（每天波动小）
- 回测的"最大回撤"= 峰值到谷值最大跌幅（趋势性下跌）

低波动股**日波动低**（每天稳），但**照样趋势性下跌**。牛市涨不动（错过主升浪），熊市跟着跌——"涨幅没吃到，回撤吃到了"。F5 单因子回撤 35.44% 看似低于 F2 的 51.6%，只是因为低波动股整体涨得少、起点低，不是真正的下行保护。

### F2+F5 组合回测：成功

| 指标 | F2-EP 单因子 | F2+F4 组合（失败） | **F2+F5 组合** |
|------|-------------|------------------|---------------|
| 总收益 | 366% | 187% | **372.71%** |
| Alpha | 0.07 | 0.18 | **0.07** |
| Sharpe | 0.34 | 0.18 | **0.37** |
| 最大回撤 | 51.6% | 57.1% | **47.62%** |

对比 F2-EP 单因子基线：
- **收益持平**：372.71% vs 366%（+6.71%，没被拖累）
- **Alpha 持平**：0.07 vs 0.07
- **Sharpe 微升**：0.37 vs 0.34（+0.03）
- **回撤下降**：47.62% vs 51.6%（**-3.98 个百分点**）

**这是所有降回撤尝试中第一次全方位不劣化。** 之前两次全部失败：
- PE 锚定择时：失败（牛市减仓代价 > 熊市减仓收益）
- F2+F4 组合：失败（收益 366%→187%，回撤 51.6%→57.1%，全方位劣化）

### 为什么 F5 单因子失败但组合成功

- F5 单因子选"最最低波动的 50 只" → 落在倒 U 型极端端 → 涨不动照样跌
- F2+F5 组合选"又便宜又稳健的 50 只"（`0.5*z(ep) + 0.5*z(vol)`）→ EP 拉动选股偏向便宜股，LowVol 在便宜股内部做风险微调 → 保住 EP 收益能力，同时低波动特性在组合层面起温和风险缓冲

**F5 的正确定位是风险调节器，不是独立选股器。** 这和 F4-ROE 不同：F4 单独是负 alpha，加到 EP 上也是负增量；F5 单独失败，但作为组合辅助因子有效。

### F5 方向判定：保留为组合辅助因子

F5-LowVol 单因子方向终止（不作为独立策略），但作为 F2-EP 的组合辅助因子保留。F2+F5 组合（等权 z-score 合成）成为当前最优策略：收益持平 F2 单因子，回撤降 4 个百分点，Sharpe 微升。

### 降回撤方向复盘（更新）

F2-EP 回撤 51.6% 的降回撤尝试结果：
- PE 锚定择时：失败（牛市减仓代价 > 熝市减仓收益）
- 叠加 F4-ROE 质量因子：失败（负 alpha，全方位劣化）
- **F2+F5 组合：成功**（收益持平，回撤 51.6%→47.62%，Sharpe 0.34→0.37）

"多因子组合降波"假设验证通过，但有关键前提：辅助因子必须与主因子**低相关**且**不拖累收益**。F4-ROE 与 EP 相关性高（都偏大盘价值），叠加是负增量；F5-LowVol 与 EP 选股池不同（便宜股 vs 稳健股），低相关，组合有效。
