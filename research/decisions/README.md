# research/decisions/

存放因子去留、权重方案、参数选择的**决策记录**。每个决策附依据与后果。

## 命名规范

`<spec_id>-decision.md`，如 `P1-F1-EV-2026Q2-v1-decision.md`。

## 决策记录模板

```markdown
# 决策 — <spec_id>

## 决策类型
<因子去留 | 权重方案 | 参数选择 | 放弃方向>

## 决策
<一句话结论>

## 依据
<数据 / 报告 / 行号>

## 后果
<对后续 Phase 的影响>

## 决策人 / 日期
```

## 约束

- 决策必须有数据支持，禁用"根据经验"（agent-workflow 抑制确认偏误第五机制）。
- 放弃方向的决策必须附"为什么继续仍合理"才能重启迭代（第六机制）。
